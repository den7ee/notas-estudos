# Sobre HTML
- Tim Berners Lee começou esboçar a linguange na década de 80, a primeira versão lançada em 1991 e a versão 5 em 2014;

# Elemento
- É a base do HTML;

- Possui uma tag de abertura, atributo, conteúdo e tag de fechamento;
```
<html>
```

# Estrutura básica
```
<!DOCTYPE html>
<html>
  <head>
    <meta>
    <title></title>
  </head>
<body>
</body>
</html>
```
